11
Recomendacion

Cual elegir?

V1 Original
Maximo detalle. Ideal si el logo se usa grande.
V2 Minimal
La G dentro. Unico, diferenciador.
V3 Bold ★
El mas versatil y potente. Funciona a cualquier tamano.
V4 Outline
Elegante, ligero. Para uso secundario.
V5 Suave
Mas amigable. Para apps consumer.
Mi recomendacion:
V3 Bold como logo principal — escudo blanco con G naranja. Contundente, limpio, funciona desde favicon 16px hasta valla publicitaria. V4 Outline como version secundaria para fondos complejos o usos decorativos.
